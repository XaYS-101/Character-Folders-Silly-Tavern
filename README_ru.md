# Character Folders

Расширение для SillyTavern, которое наконец-то позволяет организовать список персонажей так, как он того заслуживает: сворачиваемые вложенные папки с цветами, иконками, умными правилами автонаполнения и закреплением. Работает и на главном экране — полка закреплённых персонажей для быстрого запуска и группировка «Recent Chats» по вашим папкам.

**Важно:** карточки персонажей не изменяются. Вся структура папок хранится в настройках расширения — удалите расширение, и все карточки вернутся как были. (Но бэкапы всё равно делайте.)

**Примечание:** расширение активно разрабатывается и не идеально. Баги могут случаться и будут случаться. Если что-то сломалось, проверьте консоль браузера на наличие ошибок и сообщите о проблеме с подробностями.

## Функции

### 1. Система папок
Организуйте персонажей в папки, которые отображаются как сворачиваемые группы вверху списка:
- **Вложенные папки:** создавайте подпапки через выбор «Родительская папка». Глубина не ограничена (есть защита от циклов).
- **Цвета и иконки:** задайте папке цветную полосу и эмодзи/иконку для быстрой навигации.
- **Сворачивание:** переключатель у каждой папки + «свернуть/развернуть всё» в один клик. Состояние запоминается.
- **Правдивые счётчики:** в заголовке показано реальное число персонажей по всем страницам, а не только по видимой.

### 2. Раскладка персонажей (мобильный приоритет)
- 📂 **Кнопка папки:** нажмите кнопку на карточке персонажа и выберите папку из списка. Крупные тач-таргеты, удобно на телефоне.
- 🖱 **Drag-and-drop:** на ПК перетащите карточку персонажа на заголовок папки.
- 👥 **Группы персонажей:** group chats тоже можно складывать в папки наравне с одиночными.

### 3. Закрепление и solo-фильтр
- 📌 **Закрепление папок:** закреплённая папка всегда наверху, независимо от сортировки.
- 👁 **Solo-фильтр:** кнопка показывает только эту папку и прячет всё остальное; чип в панели сбрасывает фильтр.

### 4. Умные папки (автонаполнение)
Задайте папке правило — подходящие персонажи попадут в неё автоматически:
- **Есть тег…** — по нативному тегу SillyTavern.
- **Имя содержит…** — по подстроке в имени.
- **Создан после даты…** — по дате создания карточки (`ГГГГ-ММ-ДД`).

Ручная привязка персонажа всегда важнее умного правила.

### 5. Главный экран (welcome / Recent Chats)
Расширение работает и на домашнем экране SillyTavern:
- ⭐ **Полка закреплённых персонажей:** ряд персонажей для запуска в один клик сверху. Закрепить персонажа — кнопкой 📌 на его карточке.
- 🗂 **Группировка Recent Chats:** недавние чаты раскладываются по тем же папкам персонажей (по аватару чата), со сворачиванием и закреплёнными папками наверху. Чаты без папки остаются в обычном порядке ниже.
- Обе функции включаются галочками в настройках (раздел «Главный экран») и по умолчанию включены.

### 6. Массовые операции
Нажмите **Выбрать**, чтобы включить чекбоксы на карточках, затем **Переместить выбранных** — переложите всех отмеченных в папку разом.

### 7. Сортировка папок
Порядок папок вручную (стрелки `▲▼`, удобно на мобильном), либо по имени А→Я / Я→А, либо по количеству. Закреплённые папки всегда первыми.

### 8. Экспорт / импорт
Сохраняйте и переносите всю структуру папок между устройствами через **Экспорт папок…** / **Импорт папок…** — обычный JSON.

### 9. Локализация (EN / RU)
Полные английский и русский переводы. Переключите язык в настройках, и интерфейс обновится в реальном времени без перезагрузки.

### 10. Адаптивность для мобильных
Сделано с прицелом на телефоны: тач-таргеты ≥40px, иконочная панель на узких экранах, чипы полки во всю ширину, аватары крупнее на тач-устройствах.

## Известные ограничения и нюансы

- **Пагинация:** SillyTavern рендерит список персонажей постранично, поэтому физически группируются только карточки текущей страницы (счётчик в заголовке при этом правдивый — по всем страницам). Чтобы видеть все папки целиком, увеличьте размер страницы списка; группы пересобираются автоматически при прокрутке/переключении.
- **Поиск:** при активном поиске пустые папки скрываются.
- **Данные папок** хранятся в настройках расширения ST. Они переживают перезагрузку, смену персонажа и переименования, но при ручном удалении файла настроек потеряются — делайте экспорт.
- **Правило «создан после даты»** зависит от `create_date` карточки; у персонажей, загруженных «поверхностно» (не открытых в этой сессии), даты может не быть до открытия.
- **Глубина вложенности** не ограничена, но на мобильном интерфейс становится тесным после 3–4 уровней.
